## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/quran-tracker-pwa.git
```

### Enter Project Directory

```bash
cd quran-tracker-pwa
```

### Run Local Server

Python:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
